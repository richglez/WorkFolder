Práctica- No. 1.5.- Conceptos


  • R= Los sistemas distribuidos son sistemas de computación que consisten en múltiples computadoras interconectadas que trabajan juntas para lograr un objetivo común. Estos sistemas permiten que las computadoras compartan recursos, como archivos, impresoras y capacidades de procesamiento, y pueden estar ubicados en diferentes lugares físicos. Los sistemas distribuidos pueden ofrecer ventajas como escalabilidad, confiabilidad y rendimiento mejorado, pero también presentan desafíos como la coordinación y la consistencia de los datos entre las computadoras.


  • R= Es un conjunto de dispositivos electrónicos interconectados, como computadoras, servidores, dispositivos de almacenamiento y dispositivos periféricos, que pueden comunicarse entre sí para compartir recursos, datos y aplicaciones. Estas redes pueden ser cableadas o inalámbricas y permiten compartir información y servicios, como acceso a internet, impresoras y archivos, entre los dispositivos conectados. Las redes de computadoras pueden variar en tamaño y alcance, desde redes locales pequeñas en hogares y oficinas hasta redes globales que abarcan todo el mundo, como internet.


  • La comunicación simplex es un tipo de comunicación unidireccional en la que la información fluye en una sola dirección, desde el emisor hasta el receptor, pero no en ambas direcciones simultáneamente. Un ejemplo común de comunicación simplex es la radio, donde un locutor transmite información que los oyentes reciben, pero los oyentes no pueden responder directamente al locutor a menos que haya un canal separado de comunicación.


  • R= La comunicación half-duplex es un tipo de comunicación en la que la información puede fluir en ambas direcciones, pero no al mismo tiempo. En lugar de ello, la comunicación se realiza en una sola dirección a la vez, alternando entre enviar y recibir. Esto contrasta con la comunicación full-duplex, donde la información puede fluir en ambas direcciones simultáneamente.


  • R= La comunicación full-duplex es un tipo de comunicación en la que se pueden enviar y recibir datos simultáneamente en ambas direcciones. Esto significa que dos dispositivos pueden comunicarse entre sí al mismo tiempo, lo que permite una comunicación más rápida y eficiente. Es común en sistemas de telecomunicaciones y redes de computadoras donde se requiere una comunicación bidireccional continua y en tiempo real.


  • R= LA (acrónimo de "Licencia de Aplicación") se refiere a un tipo de licencia de software que otorga al usuario final el derecho de utilizar una aplicación de software de acuerdo con los términos y condiciones especificados por el proveedor del software. La licencia de aplicación suele especificar detalles como el número de dispositivos en los que se puede instalar la aplicación, las restricciones de uso, las actualizaciones y el soporte técnico disponibles, entre otros aspectos. La LA puede ser gratuita o de pago, y su objetivo es regular el uso y la distribución legal del software.


  • R= "MA" puede referirse a "Metodología Ágil". Las metodologías ágiles son enfoques para el desarrollo de software basados en principios colaborativos, iterativos e incrementales. Se centran en la entrega rápida y flexible de software funcional, adaptándose a los cambios en los requisitos y en el entorno del proyecto. Algunas de las metodologías ágiles más conocidas incluyen Scrum, Kanban, Extreme Programming (XP) y Crystal. Estas metodologías se han vuelto populares debido a su capacidad para mejorar la eficiencia, la calidad y la satisfacción del cliente en el desarrollo de software.


  • R= Una WAN (Wide Area Network o Red de Área Amplia) es una red de computadoras que cubre un área geográfica extensa, como ciudades, países o incluso a nivel mundial. Está compuesta por varios dispositivos interconectados, como routers, switches y enlaces de comunicación, que permiten la comunicación entre diferentes ubicaciones geográficas. Las WANs se utilizan comúnmente para conectar sucursales de una empresa, permitir el acceso a internet y facilitar la comunicación entre organizaciones dispersas geográficamente.


  • R= La tecnología se refiere al conjunto de conocimientos técnicos, científicos y metodológicos que se utilizan para diseñar, crear y desarrollar productos, servicios y sistemas para satisfacer las necesidades humanas. La tecnología puede abarcar una amplia gama de áreas, como la informática, la electrónica, la biotecnología, la ingeniería, entre otras, y suele estar relacionada con la aplicación práctica del conocimiento científico.


  • R= HTML (HyperText Markup Language o Lenguaje de Marcado de Hipertexto) es el lenguaje estándar utilizado para crear y diseñar páginas web. Define la estructura y el contenido de una página web mediante una serie de elementos y etiquetas que permiten organizar y presentar información en formato de hipertexto. HTML se utiliza junto con CSS y JavaScript para crear páginas web interactivas y atractivas.


  • R= CSS (Cascading Style Sheets o Hojas de Estilo en Cascada) es un lenguaje de diseño utilizado para controlar la presentación y el aspecto visual de una página web. Permite definir estilos, como colores, fuentes, márgenes, espaciado, y disposición de elementos, para mejorar la apariencia y la legibilidad de la página. CSS se utiliza junto con HTML y JavaScript para crear sitios web atractivos y funcionales.


  • R= JavaScript es un lenguaje de programación de alto nivel, interpretado y orientado a objetos, utilizado principalmente para crear páginas web interactivas y dinámicas. Permite agregar funcionalidades avanzadas a sitios web, como animaciones, efectos visuales, validaciones de formularios, interacción con el usuario, y actualización de contenido de forma dinámica sin necesidad de recargar la página. JavaScript es compatible con la mayoría de los navegadores web y es una herramienta esencial en el desarrollo web moderno.


  • R= Internet es una red de redes de computadoras interconectadas a nivel mundial que utilizan un conjunto común de protocolos de comunicación, como TCP/IP (Protocolo de Control de Transmisión/Protocolo de Internet), para permitir la transmisión de datos y la comunicación entre dispositivos. Internet proporciona acceso a una amplia gama de recursos y servicios, como páginas web, correo electrónico, transferencia de archivos, mensajería instantánea, entre otros, y es fundamental para la comunicación, la colaboración y el intercambio de información a nivel global.


  • R= La Web, abreviatura de World Wide Web o Red Mundial, es un sistema de información en línea que permite acceder y compartir contenido a través de internet. La Web se basa en la tecnología de hipertexto y utiliza el protocolo HTTP (Protocolo de Transferencia de Hipertexto) para la transferencia de datos. Permite a los usuarios acceder a sitios web y páginas web, navegar por ellas, interactuar con su contenido y acceder a servicios en línea, como correos electrónicos, redes sociales, comercio electrónico, entre otros.


  • R= La etiqueta `` se utiliza para definir el inicio y el final de un documento HTML (HyperText Markup Language) y especifica que el contenido dentro de ella es un documento HTML. Esta etiqueta es obligatoria en todos los documentos HTML y sirve como contenedor principal para todo el contenido de la página, incluyendo elementos como el encabezado, el cuerpo y el pie de página.


  • R= La etiqueta "< title > < /title>" se utiliza para definir el título de un documento HTML, que se muestra en la barra de título del navegador o en la pestaña del navegador. El título de la página es importante porque proporciona información sobre el contenido de la página y ayuda a los usuarios y a los motores de búsqueda a entender de qué se trata la página. Además, el título de la página también se utiliza como parte del resultado de búsqueda en los motores de búsqueda, por lo que es importante que sea descriptivo y relevante para el contenido de la página.
