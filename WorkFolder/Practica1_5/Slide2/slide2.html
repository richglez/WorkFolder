Práctica- No. 1.5.- Conceptos


  • R= Los sistemas distribuidos son sistemas de computación que consisten en múltiples computadoras interconectadas que trabajan juntas para lograr un objetivo común. Estos sistemas permiten que las computadoras compartan recursos, como archivos, impresoras y capacidades de procesamiento, y pueden estar ubicados en diferentes lugares físicos. Los sistemas distribuidos pueden ofrecer ventajas como escalabilidad, confiabilidad y rendimiento mejorado, pero también presentan desafíos como la coordinación y la consistencia de los datos entre las computadoras.


  • R= La etiqueta ` head ` se utiliza para contener información sobre el documento HTML que no se muestra en el navegador web. Esta información incluye elementos como el título de la página ` title `, metadatos ` meta `, enlaces a hojas de estilo ` link `, scripts ` script `, y otros elementos que afectan la presentación o el comportamiento del documento. El contenido dentro de la etiqueta ` head ` no se muestra en la página web, pero es importante para el correcto funcionamiento y la optimización de la página.


  • R= La etiqueta `body` se utiliza para definir el contenido principal de un documento HTML que se mostrará en el navegador web. Todo el contenido visible de la página, incluyendo texto, imágenes, enlaces, formularios, y otros elementos, debe estar contenido dentro de la etiqueta `body`. Esta etiqueta delimita el inicio y el final del contenido visible de la página y es una parte fundamental de la estructura de un documento HTML.

  • R= La etiqueta `h1` se utiliza para definir el encabezado más importante o relevante de una sección o página web. Por lo general, se utiliza para el título principal de la página o el encabezado de una sección importante. La etiqueta `h1` es parte de una serie de etiquetas de encabezado (`h1` a h6`) que se utilizan para organizar y estructurar el contenido de la página en diferentes niveles de importancia.


  • R= La etiqueta `!DOCTYPE ` se utiliza para declarar la versión de HTML que se está utilizando en un documento. Esta etiqueta es necesaria para que el navegador web interprete correctamente el código HTML y renderice la página de forma adecuada. El tipo de documento declarado con `!DOCTYPE ` determina qué reglas de sintaxis y qué características de HTML se pueden utilizar en el documento.

  • La etiqueta < link > en HTML se utiliza para vincular un documento con un recurso externo, como una hoja de estilo CSS, un ícono o un archivo de fuente. Su finalidad es proporcionar información adicional al navegador sobre cómo debe renderizar el documento. Por ejemplo, al vincular una hoja de estilo CSS con < link rel="stylesheet" href="estilos.css" >, el navegador sabe que debe aplicar esos estilos al documento HTML.

  • JavaScript es un lenguaje de programación que se utiliza principalmente para agregar interactividad y funcionalidad dinámica a las páginas web. Es un lenguaje de programación de alto nivel, interpretado por los navegadores web, lo que significa que no necesita compilarse antes de ser ejecutado. JavaScript se utiliza para realizar acciones como validar formularios, crear efectos visuales, manipular el contenido de la página en tiempo real, comunicarse con servidores web para cargar o enviar datos sin recargar la página completa (usando AJAX), entre otras cosas. JavaScript es un componente fundamental de las aplicaciones web modernas y es compatible con la mayoría de los navegadores web.

  • "Extensible Markup Language" (Lenguaje de Marcado Extensible, en español). Es un lenguaje de marcado similar a HTML, pero diseñado para almacenar datos de una manera legible tanto por humanos como por máquinas. XML se utiliza comúnmente para estructurar y organizar datos de manera jerárquica, utilizando etiquetas personalizadas para definir los diferentes elementos de los datos. XML es muy flexible y puede ser utilizado en una amplia variedad de aplicaciones, como el intercambio de datos entre sistemas diferentes, la configuración de aplicaciones, la representación de documentos estructurados, entre otros.

  • R= un atributo es una característica adicional que se le asigna a un elemento para proporcionarle más información o controlar su comportamiento. Los atributos se utilizan para modificar la forma en que se muestra o se comporta un elemento en una página web. Por ejemplo, en la etiqueta de imagen < img >, el atributo src se utiliza para especificar la ubicación de la imagen que se va a mostrar. Otros atributos comunes en HTML incluyen href para los enlaces < a >, class y id para la asignación de estilos CSS y la identificación de elementos, y alt para proporcionar texto alternativo en caso de que la imagen no se pueda mostrar. Los atributos están escritos dentro de la etiqueta de apertura de un elemento y se especifican en el formato nombre="valor".

  • (Structured Query Language) es un lenguaje de programación especializado utilizado para gestionar y manipular bases de datos relacionales. SQL se utiliza para realizar tareas como la creación y modificación de bases de datos, la inserción, actualización y eliminación de datos, la realización de consultas para recuperar información de una base de datos y la definición de la estructura de una base de datos, incluyendo tablas, índices y restricciones.

  • (Data Definition Language) es un subconjunto del lenguaje SQL (Structured Query Language) que se utiliza para definir la estructura y el esquema de una base de datos. Las instrucciones DDL se utilizan para crear, modificar y eliminar objetos de base de datos como tablas, índices, vistas y esquemas. Algunos de los comandos DDL más comunes incluyen:

    CREATE: se utiliza para crear nuevos objetos de base de datos como tablas, índices y vistas.

    ALTER: se utiliza para modificar la estructura de los objetos de base de datos existentes, como agregar, modificar o eliminar columnas de una tabla.

    DROP: se utiliza para eliminar objetos de base de datos existentes, como tablas, índices o vistas.

  • (Data Manipulation Language) es un subconjunto del lenguaje SQL (Structured Query Language) que se utiliza para manipular los datos almacenados en una base de datos relacional. Las instrucciones DML se utilizan para realizar operaciones como la inserción, actualización, eliminación y recuperación de datos en una base de datos. Algunos de los comandos DML más comunes incluyen:

    SELECT: se utiliza para recuperar datos de una base de datos.

    INSERT: se utiliza para insertar nuevos registros en una tabla.

    UPDATE: se utiliza para actualizar los registros existentes en una tabla.

    DELETE: se utiliza para eliminar registros de una tabla.
