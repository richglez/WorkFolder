  • 1. Escribir ¿Qué son los sistemas distribuidos?

    • Es un modelo para resolver problemas de computación masiva utilizando un gran número de ordenadores organizados en clústeres incrustados en una infraestructura de telecomunicaciones distribuida.

  • 2. Escribir definición de red de computadoras

    • Una red de computadoras, también llamada red de ordenadores o red informática, es un conjunto de equipos conectados por medio de cables, señales, ondas o cualquier otro método de transporte de datos, que comparten información (archivos), recursos (CD-ROM, impresoras, etc.)

  • 3. Escribir ¿Qué es comunicación simplex?

    • Es un modo de comunicación en el que los datos se transmiten en una sola dirección, desde un emisor hacia un receptor, sin la capacidad de enviar datos en sentido contrario. En este sentido, se entiende como una comunicación asimétrica y que no permite la retroalimentación.

  • 4. Escribir ¿Qué es comunicación half-duplex?

    • Ambos dispositivos pueden transmitir y recibir datos en los medios, pero no pueden hacerlo en forma simultánea. Ethernet ha establecido reglas de arbitraje para resolver conflictos que surgen de instancias donde más de una estación intenta transmitir al mismo tiempo.

  • 5. Escribir ¿Qué es comunicación full-duplex?

    • Ambos dispositivos pueden transmitir y recibir datos en los medios al mismo tiempo.

  • 6. Escribir la definición de LAN

    • Es una red de computadoras que permite la comunicación y el intercambio de datos entre diferentes dispositivos a nivel local, ya que está limitada a distancias cortas. Por esta razón, esta red se utiliza para hogares privados, tales como una casa o un apartamento, y en empresas

  • 7. Escribir la definición de MAN

    • Red de área metropolitana es una red de área metropolitana es una red de alta velocidad que da cobertura en un área geográfica extensa, proporcionando capacidad de integración de múltiples servicios mediante la transmisión de datos, voz y vídeo, sobre medios de transmisión tales como fibra óptica y par trenzado.

  • 8. Escribir la definición de WAN

    • Una red de área amplia, o WAN, es una red de computadoras que une e interconecta varias redes de ámbito geográfico mayor, por ejemplo redes de área local, aunque sus miembros no estén todos en una misma ubicación física.

  • 30. Escribir ¿Cuál es la finalidad de la etiqueta del?

    • se utiliza para representar texto que ha sido eliminado o marcado como obsoleto en una versión anterior de un documento, pero que aún se mantiene en el archivo por razones históricas o para referencia. El texto dentro de la etiqueta generalmente se muestra con una línea a través de él para indicar que ha sido eliminado. Esta etiqueta es útil en situaciones donde se necesita mostrar cambios o revisiones en un documento, como en un sistema de control de versiones o en documentos legales.

  • 31. Escribir ¿Cuál es la finalidad de la etiqueta tt?

Sin embargo, esta etiqueta ha sido desaprobada en HTML5 y ya no se recomienda su uso. En su lugar, se recomienda utilizar la etiqueta < code > para representar fragmentos de código o texto que debe mostrarse en una fuente de teletipo o monoespaciada, y la etiqueta < kbd > para representar entrada de teclado.

  • 35. Escribir ¿Cuál es la finalidad de la etiqueta alert?

    • Se utiliza para mostrar un mensaje emergente en el navegador. Este mensaje puede ser una notificación, advertencia, mensaje de error o cualquier otro tipo de información que el desarrollador desee comunicar al usuario. El mensaje se muestra en una ventana modal y detiene la ejecución del script hasta que el usuario hace clic en el botón de aceptar. La función alert() es útil para comunicarse de manera sencilla con el usuario, pero se debe usar con moderación para no interrumpir la experiencia de usuario.
